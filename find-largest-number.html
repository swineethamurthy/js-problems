Problem Statement:

Get the size of list of numbers. Get that many numbers from the user and display the largest number along with the entered size and numbers.

Assumptions

  • Only numbers are entered in input for the size and list of numbers. No special characters or alphabets allowed
  • No decimal numbers or fractions are allowed.Only whole number are entered
  • The size of the list can be between 0 and 10
  • The input numbers can be between the range 0 to 1000

Psuedocode

  1. Ask the size of the list from the user
  2. Get the first number
  3. Store it in an array
  4. Get the next number
  5. Add it to the array
  6. Check the largest of the two numbers and store in a separate variable
  7. Get the next number
  8. Add it to the array
  9. Compare this number and the variable and store the largest in the variable.
  10. Repeat the last three steps till the size of the list entered by the user.
  11. Display the size of the list
  12. Display the array as the result
  13. Display the largest number
