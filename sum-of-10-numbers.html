Problem Statement:

Get 10 numbers as input from user and display their sum along with the entered numbers.

Assumptions

  • Only numbers are entered in input. No special characters or alphabets allowed
  • No decimal numbers or fractions are allowed.Only whole number are entered
  • The input number can be between the range 0 to 1000

  2. Store it in an array
  3. Get the second number
  4. Include this number in the array
  5. Intialise sum to 0

  7. Add the next number to sum
  8. Repeat the above step till the 10th number
  9. Print the numbers entered by the number
  10. Print the sum
