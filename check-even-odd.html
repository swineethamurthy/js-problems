Problem Statement:

Get 10 numbers as input from user and display if they are even or odd

Assumptions

  • Only numbers are entered in input. No special characters or alphabets allowed
  • No decimal numbers or fractions are allowed.Only whole number are entered
  • The input number can be between the range 0 to 1000

Psuedocode

  1. Accept the first number from user
  2. Check if the number is divisible by 2
  3. If it is divisible by 2 store the number to an array as even along with the number
  4. Else store it to an array as odd along with the number
  5. Repeat the steps for 10 times
  6. Display the array as the result
